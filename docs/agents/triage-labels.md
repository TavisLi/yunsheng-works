# Triage 標籤

| Canonical role | GitHub 標籤 | 用途 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 等待維護者評估 |
| `needs-info` | `needs-info` | 等待提出者補充資訊 |
| `ready-for-agent` | `ready-for-agent` | 規格完整，可由代理獨立實作 |
| `ready-for-human` | `ready-for-human` | 需要人工實作或判斷 |
| `wontfix` | `wontfix` | 不會採取行動 |

工程技能提及 canonical role 時，使用表格中對應的 GitHub 標籤，不建立同義標籤。
